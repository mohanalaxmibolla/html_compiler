HTML Tutorial

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

  • <!DOCTYPE html> → Defines the document type
  • <html> → Root element
  • <head> → Metadata, title, CSS/JS links
  • <body> → Content (headings, paragraphs, images, etc.)
