HTML Basics

1

Document Structure

Every HTML document needs a basic structure:

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  <h1>My Heading</h1>
  <p>My paragraph.</p>
</body>
</html>
2

Common Tags

Essential HTML tags you should know:

<h1> to <h6> // Headings
<p> // Paragraphs
<a href="#"> // Links
<img src="#" alt="#"> // Images
<div> // Divisions
<span> // Inline container
